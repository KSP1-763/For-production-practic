# Программный модуль «РемонтПК»

## Описание

Программный модуль для автоматизации учёта технического обслуживания и ремонта компьютерной техники в ОМВД России по Ставропольскому району.

Разработан в рамках ПМ.02 «Осуществление интеграции программных модулей» для оптимизации процессов:
- Учёт компьютерной техники (ПК, ноутбуки, периферия, сетевое оборудование)
- Регистрация и обработка заявок на ремонт
- Назначение ответственного инженера и отслеживание статуса ремонта
- Учёт ремонтных операций и заменённых комплектующих
- Ведение складского учёта запчастей
- Формирование отчётности и актов списания

   Развернуть базу данных: откройте SQL Server Management Studio и выполните скрипт Database/Database.sql для создания базы данных и тестовых данных.
